voip.ms SMS to Acrobits Softphone/Groundwire

STEP 1

Enable VoIP.ms API

You'll need to log into your VoIP.ms account and visit Main Menu -> SOAP and REST/JSON API (or just click here.)

Scroll down to API Configuration and do the following:

  • Enter an API Password (usually something different than your regular account password), enter the confirmation, and click Save API Password.
  • If Enable API is disabled, click Enable/Disable API so that it shows Enabled.
  • In the text box labeled, Enable IP Addresses, enter 0.0.0.0 and click Save IP Addresses.

When finished, your screen should look something like this:

STEP 2

Open Web Services in Acrobits

Navigate your Acrobits Softphone/Groundwire app to Web Services, via the following screens:

  1. Keypad
  2. Settings
  3. Accounts
  4. Edit Account
  5. Advanced Settings
  6. Web Services

The following screenshots are from Groundwire:

Once you have the Web Services screen ready, return to this webpage and hit NEXT.

STEP 3

Activate VoIP.ms SMS

Now enter your VoIP.ms username and API password, along with your 10-digit US/Canada DID.

SMSglue will use the VoIP.ms API to activate SMS on this number, hook up push notifications, and generate a provisioning link for your device.
